Getting started

Hey there user! This documentation will show you what the prerequesites are and how to install the mod!

Prerequesites

  1. Ensure that you have the latest version of Forge (for 1.10.2, Forge version 12.18.2.2106; for 1.11.2, Forge version 13.20.0.2258; for 1.12.1, Forge version 1.12.1-14.22.1.2485 and for 1.12.2, Forge version 14.23.0.2512)

    1. If Forge is not installed, run the installer for the version of Forge as specified in the previous step.
