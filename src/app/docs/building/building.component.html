Building the source code

Prerequisites

You will need the following with you:

  • Git (should be pre-installed on Mac, downloads available here)
  • Basic knowledge of the terminal/command line (required)
  • A computer (obviously)

Instructions

To build the source code, follow these steps:

  1. Get Eclipse for your platform (Windows/ macOS) by heading here and select the Eclipse IDE for Java EE Developers option.

  2. Click on one of the expansion panels to show instructions on setting up for a specific Minecraft version:

    Minecraft 1.12 - 1.12.2 Instructions for Minecraft 1.12 to 1.12.2 (v2.x)

    Instructions for Minecraft 1.12 - 1.12.2

    Execute the following code on the command line:

    git clone https://github.com/Chan4077/First-Mod.git git checkout 1.12

    Or this method (if you're using Git 1.8.3 and above):

    git clone -b 1.12 --single-branch https://github.com/Chan4077/First-Mod.git     Minecraft 1.11.2 Instructions for Minecraft 1.11.2 (v1.2, v1.2.1)

    Execute the following code on the command line:

    Instructions for Minecraft 1.11.2

    git clone https://github.com/Chan4077/First-Mod.git git checkout 1.11.2

    Or this method (if you're using Git 1.8.3 and above):

    git clone -b 1.11.2 --single-branch https://github.com/Chan4077/First-Mod.git

    NOTE: This version is no longer supported. You can try adding more stuff to it if you want to! (You can also submit a PR!)

     Minecraft 1.10.2 Instructions for Minecraft 1.10.2 (v1.1)

    Instructions for Minecraft 1.10.2

    Execute the following code on the command line:

    git clone https://github.com/Chan4077/First-Mod.git git checkout 1.10.2

    Or this method (if you're using Git 1.8.3 and above):

    git clone -b 1.10.2 --single-branch https://github.com/Chan4077/First-Mod.git

    NOTE: This version is no longer supported. You can try adding more stuff to it if you want to! (You can also submit a PR!)

  3. Next, run this command on your command line:

    Windows Command to run for Windows gradlew.bat setupCIWorkspace gradlew.bat build

    (Click on the "Copy code" icon button to copy the code and paste it on the command line at the project's directory.)

     Mac Command to run for Mac ./gradlew setupCIWorkspace ./gradlew build

    (Click on the "Copy code" icon button to copy the code and paste it on the command line at the project's directory.)

    For more info about all the commands, view README.txt, available at the project's root directory.

  4. The path to get the JAR file will be at *. Chuck that in your Minecraft's mods folder.

Done!
